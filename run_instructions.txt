=========== Required Python Modules ===================

1. flask
2. flask-login
3. flask-sqlalchemy
4. flask-wtf
5. wtforms
6. flask-bcrypt

=================== HOW TO RUN ==========================

1. Make sure you have all the required modules installed.
2. Type "python run.py" in the terminal of current directory.
3. Open the link created on the terminal.
4. Follow the webpage .
